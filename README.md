# A 角が立たない言い換えメーカー

相手と「言いたいこと」を選ぶ／検索すると、角が立たない例文が「やわらか／標準／きっぱり」の3トーンで出るツール。データ駆動の単一HTML（バニラJS・**file:// 直開きOK**）。

> 🟢 **一般公開中：https://chiiita.github.io/iikae/** （リポジトリ `chiiita/iikae` Public・GitHub Pages）
> 📘 **別チャットで続ける人へ → まず [`HANDOVER.md`](HANDOVER.md) を読む**（現状・スキーマ・デザイン・公開/更新手順・次の候補すべて記載）。デプロイ/OGP生成スクリプトは `_scripts/`。

## 使い方
`index.html` をダブルクリックでブラウザで開くだけ。サーバー不要・登録不要。

## 構成
```
A_言い換えジェネレーター/
├── index.html        … UI＋ロジック（全部入り）
└── data/phrases.js   … 例文データ（ツールの中身。ここを足すだけで拡張可）
```

## 仕組み
- 現在 **155シチュエーション** 収録。各エントリに「✗言いがちNG例」＋3トーン。
- **相手12カテゴリ**：上司17／同僚・部下16／取引先14／夫・妻16／義実家14／親15／ママ友・近所15／友人16／恋人9／自分の子ども9／先生・学校7／店員・窓口7。
- **場面6種**：断る／不満・指摘を伝える／お願いする／謝る・修復する／距離を置く／感謝・ほめる。
- 入力欄は**検索ボックス**（言いたいことで絞り込み・類似マッチ：活用形/かな/送りがな違いを吸収）。`kw` で各項目に検索語を付与。
- トーンは「伝える強さ」3段階（やわらか／標準／きっぱり）で、断り以外（お願い・感謝・子への声かけ）でも自然に機能。
- **Tier1改善済み（全ブラウザ完結）**：①ヒーロー直下のグローバル検索＋よく使う候補チップ（全相手横断）②★お気に入り＋🕐最近使った（localStorage `iikae.fav`/`iikae.recent`）③カード内「自分の事情ひと言」→ 最初の◯◯に即反映 ④各トーンに**コピー**＋**送る**（navigator.share→LINE等）⑤送る前の安心コピー。
- 本文中 `｛…｝`/`{…}` は金の点線プレースホルダ表示（中身が「理由」or空なら ◯◯）。LINE向きの文には📱バッジ。

## 設計の品質基準（データを足すとき守る）
1. **I主語徹底**（「あなたは〜」断定NG → 「私は／私の手元では」）
2. **クッション言葉は1つまで**（重ねると慇懃無礼）
3. **結論(YES/NO)は必ず含む**（曖昧な返事は後で角が立つ）
4. **断り≠謝罪**（誘いを断るのは「ありがとう＋残念」で十分。むやみに謝らない）
5. トーンは「ていねいさ」でなく「**関係を続けたい度／NOの固さ**」の段階

## 拡張方法
`data/phrases.js` の `PHRASES` 配列に
`{id, person, scene, title, ng, soft, standard, firm, line}` を追記するだけ。
相手・場面を増やすなら `PERSONS` / `SCENES` に1行追加。

## 競合・差別化（2026-06リサーチ）
直接競合は実質ゼロ。汎用AI言い換え（文を持つ前提）・記事（探す手間）・アサーションアプリ（学ぶ前提）の隙間。
